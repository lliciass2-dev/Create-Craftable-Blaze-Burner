# Create: Craftable Blaze Burner
Uses KubeJS to add a custom item and custom recipes to craft Blaze Burners

**Tested on MC 1.21.1 using NeoForge 21.1.231**

## Dependencies
* Create 6.0.10
* KubeJS 2101.7.2-build.368
* KubeJS Create 2101.3.1-build.18

## Adding to KubeJS 
If you are unfamiliar with KubeJS set up run the game first to generate the KubeJS directories. Then copy the files into the respective directories. Includes a **startup script** to add the item and a **server script** to add the recipes. Model and texture for the custom item are in the **asset** directory.
